Odin-Library

This project uses HTML, CSS, and JavaScript to create a webpage for a simply library. This project was created for the Odin Project with the purpose of practicing skills with JavaScript objects.

This project contains:
- A sidebar with information about the library
- A button in the sidebar that opens a dialog
- A dialog with a form that gets user data to create a boook
- A main content section that can contain tabs displaying book information

Concepts used for this project:
- JavaScript objects
- Object constructors
- Object prototypes and prototype functions
- HTML dialogs
- HTML forms
- Flexbox
- Grid
